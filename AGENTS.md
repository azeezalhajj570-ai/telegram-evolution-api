## Project Constitution

This project is governed by the constitution at `.specify/memory/constitution.md`.
All technical decisions MUST comply with its principles:
- **I. Code Quality** — maintainable, reviewed, typed, linted code
- **II. Testing Standards** — mandatory contract/integration/unit tests, red-green-refactor
- **III. User Experience Consistency** — uniform design, explicit edge cases, documented
- **IV. Performance Requirements** — measurable budgets, benchmarks, no N+1
- **V. Security & Observability** — input validation, structured logging, health endpoints

Review the full constitution for governance, amendment, and compliance rules.
